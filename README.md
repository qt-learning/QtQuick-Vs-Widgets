# Qt Quick vs. Qt Widgets
This is the repo for the Qt Quick vs. Qt Widgets course on Qt Academy.

# Getting Started
The files provided are the demo apps created during the course. They provide a reference for the workflows covered in the course section.

# Instructions
To get started with this project, you will need to have Qt installed from the qt.io website. We Recommend using the QtCreator IDE in order to follow along with the content in the videos.

QuickVsWidgets_Section2a: Example Qt Widgets project using basic widgets. Tested with Qt 6.8.4 MinGW 64-bit.

QuickVsWidgets_Section2b: Example Qt Widgets project using the Qt Widgets Designer. Tested with Qt 6.8.4 MinGW 64-bit.

QuickVsWidgets_Section3a: Example Qt Quick project using basic Qt Quick components and Qt Quick Controls. Tested with Qt 6.8.4 MinGW 64-bit.

QuickVsWidgets_Section3b: Example Qt Quick project using Qt Design Studio. Tested with Qt 6.8.2 MSVC 2022 and Qt Design Studio 4.7.0

QuickVsWidgets_Section5: Example Qt Widgets project that incorporates a Qt Quick scene using QQuickView and QQuickWidget. Tested with Qt 6.8.4 MSVC2022 64-bit.
